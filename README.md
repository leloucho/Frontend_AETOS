# Aetos Frontend

Minimal Angular 17+ frontend scaffold with Tailwind configuration.

Install dependencies:

npm install

Run dev server (example using Vite + Angular integration or standard Angular CLI):

npm run start

Notes:
- Add components: Login, Register, Dashboard, QR Scanner, Admin Panel.
- Implement `AuthService` to manage JWT and call backend `/api/auth` endpoints.
- For production deploy to Vercel, build static assets and point to backend API.
